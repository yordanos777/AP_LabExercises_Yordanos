CHAT APPLICATION

Description:
This is a simple chat application developed using JavaFX and Socket Programming.
The project allows communication between multiple clients and the server.

Concepts Used:
- OOP Fundamentals
- Network Programming
- Multithreading
- File Handling
- JavaFX GUI

Files:
- ServerFX.java
- ClientFX.java
- Message.java

Features:
- Server and client communication
- Real-time chat
- Message timestamps
- Save chat history to chat.txt
- GUI interface using JavaFX

How to Run:
1. Run ServerFX.java
2. Run ClientFX.java
3. Send messages between client and server
